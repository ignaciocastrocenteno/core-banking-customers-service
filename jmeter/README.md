# JMeter Performance Tests

This directory contains Apache JMeter test plans used to evaluate
the performance and stability of the SOAP Customer Service.

## Structure

- test-plans/: JMeter .jmx files
- data/: CSV input datasets
- results/: Execution results (excluded from version control)

## Execution

1. Open Apache JMeter
2. Load the desired test plan from test-plans/
3. Configure target host and port if needed
4. Execute and analyze results
